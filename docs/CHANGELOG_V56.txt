Версия v56

Что добавлено:
- реестр олимпиадных предметов с кафедрой, активностью и комментарием;
- автозагрузка базового перечня предметов ВСОШ из файла data/olympiad_subjects_vsoh.xlsx;
- в импорт сессий добавлена кафедра;
- в несопоставленные строки добавлены причина, этап, признак «выбыл», комментарий;
- улучшено сопоставление предмета, кафедры и учителя при импорте;
- в реестре олимпиадников добавлены фильтры по учителю и кафедре;
- ручное изменение учителя только из списка нагрузки по предмету/кафедре;
- массовое назначение учителя в реестре;
- раздел «Несопоставленные результаты»;
- ссылки на олимпиады и реестр предметов в служебных ссылках на главной.

Технически:
- новые поля добавляются автоматически при запуске через runtime schema upgrade;
- для первичной загрузки перечня можно использовать scripts/seed_olympiad_subjects.py.
