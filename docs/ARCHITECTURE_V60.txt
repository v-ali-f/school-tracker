РЕФАКТОРИНГ СТРУКТУРЫ ПРОЕКТА
============================

Что сделано:
- добавлен слой app/core для конфигурации, БД и безопасности;
- добавлен слой app/modules, где каждый раздел получил собственный пакет;
- регистрация blueprint теперь идет через единый реестр app/modules/__init__.py;
- добавлен shared-слой для будущего безопасного переноса моделей;
- legacy-файлы app/*.py сохранены, поэтому обновление остается совместимым.

Что это дает:
- проект легче расширять;
- новые версии проще развивать без хаоса;
- можно постепенно переносить код из legacy-файлов в модульную структуру.
