Версия v56 — Олимпиады / ВСОШ

Что добавлено:
1. Реестр олимпиадных предметов с кафедрой, активностью и комментарием.
2. Базовая загрузка перечня предметов ВСОШ из файла «Предметы ВСОШ.xlsx».
3. Улучшенный импорт: поддержка и Excel, и ZIP, автоматическая подстановка кафедры из реестра.
4. Несопоставленные строки теперь сохраняются не только по ученикам, но и по предметам.
5. В реестре олимпиадников список учителей ограничивается предметом/кафедрой/годом.
6. Добавлено массовое изменение учителя.
7. На главной странице появились ссылки:
   - Реестр предметов (школьные предметы)
   - Реестр олимпиадных предметов
   - Несопоставленные результаты

Важно:
- Проект использует best-effort runtime upgrade схемы БД в app/__init__.py.
- Для уже существующей базы после запуска приложения новые nullable-поля добавятся автоматически.
- Для полной фиксации структуры можно дополнительно выполнить SQL из файла v56_manual_sql.sql.
